# Gold Market Analysis Dashboard

## 📊 Overview

This project presents an interactive analysis of gold price trends, volatility, and trading signals using Tableau.

## 🔍 Key Insights

* Long-term upward trend in gold prices
* Volatility highlights risk and opportunity periods
* Market signals indicate majority consolidation phases

## 🛠 Tools Used

* Tableau Public
* Excel (Data Preparation)
